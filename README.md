# Flat Flags

A set of flags for CSS and LESS

## Install with Bower

    bower install flat-flags

## Usage

```css
<i class="flag-sm flag-ES"></i>
```

If you use LESS you can define the path for the sprites with `@flat-flags-sprites-path`
